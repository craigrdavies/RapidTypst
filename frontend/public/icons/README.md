# Rapid Typst Desktop - Icon Placeholder

Replace these placeholder files with actual icons:

- 32x32.png - 32x32 pixel PNG icon
- 128x128.png - 128x128 pixel PNG icon  
- 128x128@2x.png - 256x256 pixel PNG icon (for Retina displays)
- icon.icns - macOS icon bundle
- icon.ico - Windows icon file

You can generate these from a single high-resolution image using tools like:
- https://icon.kitchen/
- https://www.npmjs.com/package/png-to-ico
- macOS: `iconutil` command

Recommended source image: 1024x1024 PNG with transparency
